# 🛩️ 3D飞行射击游戏

一个使用Three.js开发的3D飞行射击游戏，包含坦克目标、炸弹系统和记分功能。

## 🎮 游戏特色

- **逼真的3D飞行体验**：使用Three.js WebGL渲染引擎
- **战斗系统**：机炮射击 + 炸弹投掷
- **动态环境**：森林地形、天气系统
- **记分系统**：击毁坦克获得分数

## 🕹️ 操作控制

- **WASD/方向键**：控制飞机飞行
- **鼠标左键**：机炮射击
- **空格键**：投炸弹
- **Q/E键**：飞机滚转
- **T键**：切换天气
